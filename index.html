User Manual

FormCheck helps you compare your form against a reference video using overlay, AI skeleton analysis, and annotations.

1. Setup

  • Upload: Select a Reference Video (Video 1) and your Attempt (Video 2). You can use just one video if desired.
  • Opacity: Use the top slider to fade Video 2 in/out.

2. Adjusting Videos

  • Move (✥): Click this icon for a video, then drag on the canvas to move it, or pinch to resize/zoom it.
  • Trim ([ ]): Move the timeline slider to a point and click [ to set Start or ] to set End.
  • Bone (🦴): Toggle AI Stick Figure overlay.
  • Person (👤): Toggle AI Background Removal to cleanly overlap videos.

3. Annotation Tools

  • Draw (✏️): Select a color and draw lines.
  • Text (T): Tap to add text. Drag to move, pinch to resize.
  • Undo (↩): Remove the last action.

4. Saving

  • Save (💾): Automatically plays your videos from start to finish and downloads a .webm video file.
